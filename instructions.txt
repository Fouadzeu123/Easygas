Tu es un développeur expert en Laravel, Inertia.js, Vue 3, TailwindCSS.
Tu dois développer une application appelée EasyGas en travaillant dans un projet déjà existant Laravel + Inertia + Vue.

L'application doit être conçue en MOBILE FIRST, comme une vraie application mobile (style Uber, Bolt, Orange Money, Glovo).
Ce n’est PAS un site web classique.

========================================
📱 DESIGN MOBILE FIRST (TRÈS IMPORTANT)
========================================
L’interface doit ressembler à une application mobile :

- Pas de header web classique
- Pas de footer web classique
- Utiliser une Bottom Navigation (barre en bas)
- Interface centrée avec largeur max (max-width: 420px) sur PC
- Sur PC : l’application doit être centrée comme un téléphone
- Coins arrondis
- Ombres douces
- Animations fluides
- Style application moderne

Structure principale :
